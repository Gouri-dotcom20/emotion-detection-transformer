# emotion-detection-transformer
# Emotion Detection from Text using Transformers

## Project Overview
This project implements an automated emotion detection system using a transformer-based deep learning model (BERT). The system detects emotions such as joy, sadness, anger, love, fear, and surprise from English text.

## Features
- Transformer-based emotion classification
- Text preprocessing and normalization
- Streamlit frontend for real-time prediction
- Confidence score for predictions

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Streamlit

## How to Run
1. Install dependencies:
   pip install streamlit torch transformers
2. Run the application:
   streamlit run app.py
